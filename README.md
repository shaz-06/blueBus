# 🚍 BlueBus

A modern **Bus Ticket Booking Web Platform** inspired by RedBus & MakeMyTrip.

BlueBus allows users to search buses, view routes, check seat availability, and simulate ticket booking with a clean UI and smooth user experience.

---

## ✨ Features

🔎 **Smart Bus Search**
- Search buses by source and destination
- Instant results with available routes

🪑 **Seat Selection System**
- Visual seat layout
- Select and reserve seats easily

📅 **Trip Planning**
- Weekend trips
- Special routes
- Travel suggestions

📄 **E-Ticket Generation**
- Ticket preview
- Downloadable ticket interface

💬 **Customer Support**
- Chat support page
- Email help page

📊 **Dynamic Pages**
- Bus tables
- Ticket information
- Cancellation policies

📱 **Responsive UI**
- Works on Desktop
- Tablet
- Mobile devices
